# 生猪价格研究数据源总结

- `ASF` 相关源本轮暂时略过

## 当前总体判断

1. **河南省高频生猪出场价公开源仍然不理想。**
   `Mysteel` 和 `搜猪网` 偏“今日价 + 一年走势图”，不足以支撑长历史原始序列抓取。

2. **目前最扎实的主数据源，是农业农村部生猪产品数据专题。**
   该专题从 **2021 年起** 已确认有较完整的月度数据，并且可以导出表格。

3. **公开官方的低频补充源是有价值的，但适合做补充和校验，不适合投入太多抓取成本。**
   例如河南发改委、河南/郑州粮储系统、统计公报类。

## 现阶段可直接进入首轮抓取的来源

### A 组：应优先拿原始数据

1. **农业农村部生猪产品数据专题**
   - 原因：字段全、时间连续、可导出。
   - 当前价值最高。

2. **农业农村部监测预警栏目“畜产品和饲料集贸市场价格情况”**
   - 原因：周度稳定，能同时覆盖生猪、仔猪、玉米、豆粕。
   - 适合作为公开官方周度主链。

3. **河南省发展改革委价格监测 / 食品价格**
   - 原因：虽然低频，但河南本地相关性高，适合做校验底稿。

### B 组：可抓，但放到第二顺位

1. **郑州市粮食和物资储备局粮油市场**
2. **河南省粮食和物资储备局相关分析文**
3. **农业农村部“国内外农产品市场动态”**

### 1. 已确认最强主源

- `农业农村部生猪产品数据专题`
  - 覆盖：2021-至今
  - 特点：月度、表格可导出、字段完整

### 2. 已确认最强周度公开源

- `农业农村部监测预警栏目周报`
  - 覆盖：近年周报稳定
  - 特点：周度、正文表格、适合抓全国生猪/玉米/豆粕/仔猪

### 3. 已确认河南本地校验源

- `河南省发改委价格监测/食品价格`
  - 覆盖：2020-07 起较多，2022 起更稳定
  - 特点：本地相关性强，但低频、非标准结构
